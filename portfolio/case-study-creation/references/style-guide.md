# Writing Style Guide

## Voice & Tone
- Confident but not boastful. Direct and clear. Shows self-awareness. 
- Casual tone to highlight my laid-back personality, but don't over-do it.
- Avoid jargon for jargon's sake but doesn't shy away from technical specificity when it matters.


## Formatting Preferences
- Use short paragraphs (2-4 sentences)
- Use pull quotes for key insights
- Section headers should be descriptive, not clever


## Things to Avoid
- Don't start with "I redesigned X for Y company"
- Don't over-explain basic design concepts
